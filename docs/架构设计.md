### 1.2 技术架构分层

| 层级 | 技术选型 | 职责 |
|------|----------|------|
| **接入层** | Nginx | 反向代理、负载均衡、静态资源 |
| **注册配置** | Nacos | 服务发现、动态配置、服务治理 |
| **业务层** | Spring Boot + MyBatis Plus | 业务逻辑处理 |
| **认证授权** | SaToken | 无状态Token认证、权限控制 |
| **服务调用** | OpenFeign | 声明式HTTP客户端、负载均衡 |
| **流量防护** | Sentinel | 流量控制、熔断降级、系统保护 |
| **数据层** | MySQL + Redis + ES + MinIO | 数据存储和检索 |
| **消息层** | Kafka + RocketMQ | 异步消息处理、削峰填谷 |
| **AI层** | OpenAI + Ollama + 意图识别 | 多模型智能服务 |
| **日志层** | ELK (Logstash + Elasticsearch + Kibana) | 日志采集、存储、可视化 |
| **分析层** | Hadoop + Spark + Hive | 大数据分析 |

## 四、服务架构

### 4.1 服务依赖关系

```
Frontend
    │
    ▼
Nginx
    │
    ▼
Backend (Spring Boot)
    │
    ├──→ Nacos (服务注册/配置中心)
    ├──→ MySQL (主数据库)
    ├──→ Redis (缓存)
    ├──→ MinIO (对象存储)
    ├──→ Kafka (消息队列)
    ├──→ RocketMQ (消息队列)
    ├──→ Elasticsearch (搜索引擎)
    ├──→ OpenAI / Ollama (AI多模型)
    └──→ Logstash (日志采集) → ES → Kibana
```

### 4.2 服务通信

| 通信方式 | 使用场景 | 技术实现 |
|----------|----------|----------|
| HTTP REST | 同步调用 | OpenFeign |
| Nacos Discovery | 服务发现 | Spring Cloud Alibaba Nacos |
| Kafka | 异步消息(订单/行为) | Spring Kafka |
| RocketMQ | 异步消息(秒杀/库存) | RocketMQ Spring Boot Starter |
| SSE | 流式输出 | WebFlux |
| WebSocket | 实时通信 | Spring WebSocket |

### 4.3 服务治理

| 功能 | 实现方式 | 说明 |
|------|----------|------|
| 服务注册 | Nacos | 服务发现与健康检查 |
| 配置中心 | Nacos | 动态配置下发 |
| 负载均衡 | Spring Cloud LoadBalancer | 客户端负载均衡 |
| 熔断降级 | Sentinel | 流量防护、服务保护 |
| 服务调用 | OpenFeign | 声明式HTTP客户端 |
| 认证鉴权 | SaToken | 无状态Token认证 |

## 五、数据流架构

### 5.1 订单流程

```
用户下单
    │
    ▼
创建订单 (MySQL)
    │
    ▼
发送订单事件 (Kafka)
    │
    ├──→ 扣减库存
    ├──→ 发送通知
    └──→ 记录日志
```

### 5.2 搜索流程

```
商品变更 (MySQL)
    │
    ▼
发送变更事件 (Kafka)
    │
    ▼
同步到Elasticsearch
    │
    ▼
更新搜索索引
```

### 5.3 推荐流程

```
用户行为 (浏览/收藏/购买)
    │
    ▼
记录行为数据 (MySQL)
    │
    ▼
发送行为事件 (Kafka)
    │
    ▼
更新推荐模型 (Spark MLlib)
    │
    ▼
生成推荐结果 (Redis缓存)
```

### 5.4 AI对话流程

```
用户发送消息
    │
    ▼
意图识别 (IntentDetector)
    │
    ├──→ 商品咨询 → 获取商品信息
    ├──→ 订单查询 → 获取订单信息
    └──→ 通用问答 → 知识库匹配
    │
    ▼
获取对话历史 (Redis)
    │
    ▼
构建上下文 (商品/订单/用户)
    │
    ▼
路由LLM (OpenAI / Ollama本地Agent)
    │
    ▼
返回流式响应 (SSE)
    │
    ▼
保存对话历史 (Redis)
```

### 5.5 秒杀流程

```
用户发起秒杀
    │
    ▼
检查秒杀时间段
    │
    ▼
检查是否已购买 (一用户一商品一次)
    │
    ▼
Redis预减库存 + 数据库原子扣减
    │
    ▼
发送秒杀订单事件 (RocketMQ: seckill-order-topic)
    │
    ▼
异步创建订单 / 发送通知
```

### 5.6 签到流程

```
用户签到
    │
    ▼
检查今日是否已签到
    │
    ▼
计算连续签到天数
    │
    ▼
计算积分 (1-7分，连续天数越多积分越高)
    │
    ▼
写入签到记录 (MySQL)
    │
    ▼
返回签到结果 (积分+连续天数)
```

### 5.7 优惠券流程

```
用户领取优惠券
    │
    ▼
检查优惠券库存
    │
    ▼
检查是否已领取
    │
    ▼
扣减库存 + 用户绑定
    │
    ▼
下单时使用
    ├──→ 检查满足条件金额
    ├──→ 计算优惠后金额
    └──→ 标记优惠券已使用
```

### 5.8 文件上传流程

```
用户上传图片
    │
    ▼
后端接收 MultipartFile
    │
    ▼
调用 MinioService 上传
    │
    ▼
MinIO 生成对象URL
    │
    ▼
返回访问URL (可切换阿里云OSS)
```

## 六、安全架构

### 6.1 认证授权

```
用户登录
    │
    ▼
验证用户名密码 (BCrypt)
    │
    ▼
SaToken生成Token
    │
    ▼
返回Token给客户端
    │
    ▼
客户端携带Token请求 (Header: Authorization)
    │
    ▼
SaToken拦截器验证Token
    │
    ▼

    │
    ▼
```

### 6.2 安全策略

| 安全措施 | 实现方式 | 说明 |

| 密码加密 | BCrypt | 不可逆加密 |

| 限流防护 | Sentinel | 接口限流 |
| 分布式会话 | Redis | Token存储于Redis |
