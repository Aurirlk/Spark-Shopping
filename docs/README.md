# Shopping 电商平台

> **版本**: v2.0  
> **更新日期**: 2026-06-24  
> **技术栈**: Spring Boot + Vue 3 + MySQL + Redis + Kafka + Elasticsearch + Spark + LLM

---

## 项目简介

Shopping 是一个完整的电商平台解决方案，包含：

- **C端用户前台**: 商品浏览、购物车、订单管理、AI客服
- **B端商家后台**: 商品管理、订单管理、数据大屏
- **大数据平台**: Hadoop/Hive/Spark 数据分析
- **AI智能服务**: LLM智能客服、个性化推荐
- **搜索引擎**: Elasticsearch 全文搜索

---

## 核心特性

| 特性 | 说明 | 技术实现 |
|------|------|----------|
| 🛒 **电商核心** | 商品、订单、购物车、用户中心 | Spring Boot + MyBatis Plus |
| 🔍 **智能搜索** | 全文搜索、中文分词、高亮显示 | Elasticsearch + IK分词器 |
| 🤖 **AI客服** | 智能问答、流式输出、上下文记忆 | OpenAI API / 本地LLM |
| 🎯 **个性推荐** | 协同过滤、热门推荐、相似商品 | 基于用户的协同过滤算法 |
| 📊 **数据大屏** | 销售统计、用户分析、趋势展示 | ECharts + Vue 3 |
| 🚀 **事件驱动** | 订单事件、行为追踪、异步处理 | Kafka 消息队列 |
| ⚡ **高性能** | 缓存优化、分布式锁、连接池 | Redis + Lettuce |
| 🐳 **容器化** | 一键部署、服务编排、健康检查 | Docker Compose |

## 快速开始

### 环境要求

| 软件 | 版本 | 说明 |
|------|------|------|
| Docker | 20.10+ | 容器化部署 |
| Docker Compose | 2.0+ | 服务编排 |
| JDK | 1.8+ | 后端运行 |
| Node.js | 18+ | 前端构建 |

### 一键部署

```bash
# 1. 克隆项目
git clone <repository-url>
cd Shopping

# 2. 配置环境变量（可选）
cp .env.example .env
# 编辑 .env 文件，配置 LLM API Key

# 3. 启动所有服务
docker-compose up -d

# 4. 访问服务
# 前端首页: http://localhost:3000
# 数据大屏: http://localhost:3000/admin/data-screen
# 后端API: http://localhost:8080
```

### 服务状态

```bash
# 查看服务状态
docker-compose ps

# 预期输出
shopping-mysql          ✅ (healthy)
shopping-redis          ✅ (healthy)
shopping-zookeeper      ✅ (healthy)
shopping-kafka          ✅ (healthy)
shopping-elasticsearch  ✅ (healthy)
shopping-backend        ✅ (healthy)
shopping-frontend       ✅ (healthy)
```
